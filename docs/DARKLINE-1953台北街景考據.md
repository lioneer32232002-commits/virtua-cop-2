# 《暗線 / DARKLINE》首部曲 — 1953 台北街景・地圖考據

> 產出日期：2026-06-17 ｜ 性質：**美術／關卡考據參考**（用戶要求：街景與地圖盡量貼當年實況，**路名可虛擬、建築與街區風格要對**）。承 [市場研究](DARKLINE-市場研究報告.md) §6、[首部曲規劃](DARKLINE-首部曲規劃.md) §3。
> 用法：①餵 Gemini 生貼圖的視覺咒語庫；②設計 rail 路徑與自由段巷弄時的「走過幾條街會看到什麼」對照表。
> **史實考據；遊戲路名虛構、不指名真實機關。下標每個建築都註記「1953 是否已存在」以防時代錯置。**

---

## 0. 一句話定位：1953 台北＝「日本骨、中華魂、美國味」三層疊街

戰後第八年的台北，視覺上是三層疊加：
1. **日治的骨**——官廳石造／辰野式紅磚、榮町銀座的現代百貨、騎樓街屋（街景主結構幾乎全是日治留下的）。
2. **戰後的魂**——直式中文招牌蓋掉日文、上海來的紅包場與小吃、難民棚屋、三輪車、曬衣繩、攤販。
3. **美國的味**——中山北路的美軍酒吧／舶來品／英文招牌／吉普車（冷戰前線的點綴）。

**做對這三層的疊加感，就對了當年的台北。** 切忌做成「純古中國」或「純日本」。

## 1. 五大街區（每區＝一種建築語彙＋商店類型，可對應關卡段落）

### A. 城內（城中）＝官廳區・權力的臉
- **建築語彙**：清代台北城拆牆後的三線道環抱；區內是**日治官廳建築群**——巨大對稱的石造／辰野式紅磚白飾帶、帝冠樣式屋頂。莊嚴、冷、留白多。
- **代表建築（1953 皆在）**：總督府（→今總統府，1919）、總督府博物館（新公園內，→今台博館）、台灣銀行、地方法院、台北郵局、新公園（→今 228 公園）。
- **商店類型**：少商店、多官署／銀行／書局；安靜、肅殺。
- **遊戲用途**：**開場簡報（西緣閣樓）＋「內勤科」官署＋查封公文的紙本軌跡**都在這——權力與官僚的視覺。

### B. 榮町（衡陽路一帶）＝「台北銀座」・最繁華的商業騎樓
- **建築語彙**：連續**騎樓**商業街（頂蓋走廊＋柱列）、現代主義立面、霓虹／手繪招牌密集。
- **代表建築（1953 皆在）**：**菊元百貨**（1932，台灣第一家百貨，六層＋頂樓瞭望＝「七重天」）；衡陽路（日治「榮町通」）、寶慶路、博愛路。
- **商店類型**：鐘錶行、銀樓、洋品、書店、咖啡、相館——高檔購物街。
- **遊戲用途**：**Arena 1 騎樓街**＝最熱鬧、最多人、最「繁華中爆發槍戰」的反差。

### C. 大稻埕／迪化街＝傳統台灣商街・台灣魂的心臟
- **建築語彙**：**洋樓街屋**——兩層為主、紅磚＋木材、**巴洛克山牆／南洋殖民地式拱廊／閩南式**混合立面；女兒牆嵌店號匾額、三開拱窗、騎樓拱廊。最有「老台灣」味的一區。
- **代表建築（1953 皆在）**：迪化街街屋群（清 1850s 起）、新芳春茶行（1934）、有記名茶、霞海城隍廟、大稻埕碼頭（淡水河）。
- **商店類型**：**茶行、南北貨、中藥行、布行／紡織**（1950s 紡織新興，新光吳火獅等發跡於此）；騎樓下堆貨、麻布袋、藥櫃、曬茶。
- **歷史共鳴**：大稻埕自日治起就是**台灣本土意識／文化政治運動的搖籃**（文化協會一帶）——對劇情的「台灣魂」暗線是絕佳場景。
- **遊戲用途**：**Arena 2 市場／巷口（老聶在此被帶走）＋自由段調查巷弄**＝人最密、最有生活感、最適合「死信箱／跟監」的迷宮。

### D. 西門町＝娛樂區・霓虹與電影
- **建築語彙**：戲院群、**手繪電影海報牆**、咖啡店、上海式紅包場；西門紅樓（1908 八角樓，1953 在）。
- **代表（1953）**：武昌街一帶 10+ 戲院（1950s 起大量開）；漢口街／峨眉街／西寧南路的上海式紅包場夜總會。
- **商店類型**：戲院、咖啡廳、唱片、冰果室、攤販。年輕、流行、夜生活。
- **遊戲用途**：自由段的**情報接頭點（咖啡店／戲院後巷）**＝諜報片最愛的人潮掩護。

### E. 中山北路＝美軍區・冷戰的味道
- **建築語彙**：林蔭大道、美式 bar／餐廳／教堂、舶來品店、英文招牌、軍區大門與方向牌。
- **代表（1953）**：MAAG 周邊、美軍宿舍、西式餐廳（如轉角的 Rose Marie）、美而廉。
- **商店類型**：酒吧、西餐、舶來品、裁縫（做美式西裝）。穿制服士兵與平民混雜。
- **遊戲用途**：**Arena 3／Boss（碼頭與美軍區外圍）**＝西緣（CIA 前線）與美方的地盤、最終交火。

> （備選）**艋舺／萬華**：龍山寺、老市場、青草／蛇店＝更草根的老城；若需第二個傳統區可用。

## 2. 走過幾條街會看到什麼（建議路線＝對應六拍）

> rail 路徑與自由段動線可沿這條「由官到商、由華到洋、終於河港」的軸線走，**每過一個街區換一種建築語彙**，玩家自然讀到城市的階層：

```
城內官署（冷・石造）   →  榮町騎樓（繁華・百貨霓虹）  →  大稻埕迪化街（老台灣・洋樓茶行藥行）
   ↑開場/內勤科              ↑Arena 1 騎樓街               ↑Arena 2 市場（老聶被帶走）＋自由段巷弄
                                                                    │
        將軍玫瑰（郊宅軟禁・結尾） ← 中山北路美軍區（洋・酒吧吉普） ← 大稻埕碼頭/淡水河（倉庫・駁船）
              ↑結尾                      ↑Arena 3 外圍                  ↑Boss 交接點
```

**過街換景的具體節奏範例**（自由段巷弄）：迪化街口（藥行麻袋藥香）→ 轉進窄巷（曬衣繩、後門、水溝蓋）→ 城隍廟埕（攤販、香煙）→ 河溝邊（茶箱堆、駁船）→ 倉庫區（鐵皮、麻繩、貨堆）。

---

## 3. 共通街道元素（每個街區都鋪這些＝年代感的底）

- **交通**：三輪車（政府配發、帆布頂）、ROC 司機開的美軍吉普、腳踏車、少數公車與私家車、**冰淇淋腳踏車攤**（白箱＋遮棚）。
- **基礎設施**：水泥側溝＋剖半竹片／水泥蓋板、水泥垃圾箱、電線桿與交錯電線、**騎樓柱列**。
- **招牌**：**直式中文**手繪店招為主、殘存日文招牌（部分被蓋）、中山北路的英文招牌、戲院手繪海報。
- **生活**：曬衣繩、攤販、藤椅、招貼、反共標語布條（時代背景，**慎用、低調**）。
- **服裝**：旗袍與西式洋裝並存、美軍卡其、國軍軍裝、平民布衫／工人服。

## 4. 時代錯置紅線（這些 1953 還沒有／已沒有，別放）

| 別放 | 原因 |
|---|---|
| **中華商場**（中華路八棟） | **1961 才建**；1953 那裡是**鐵道旁的棚屋違建／難民棚戶**——要做就做棚屋，不是商場 |
| **圓山大飯店**現貌（紅宮殿） | 標誌性建築是 **1973**；1952 僅前身小館，別放紅色宮殿 |
| 現在的台北車站 | 1953 是**日治第三代站體（1941）**，非現代站 |
| 國父紀念館（1972）／中正紀念堂（1980） | 都太晚 |
| 台灣鐵道飯店 | **1945 美軍轟炸燒毀**，1953 已不在 |
| LED／電子看板／塑膠招牌 | 一律手繪／木牌／霓虹燈管 |
