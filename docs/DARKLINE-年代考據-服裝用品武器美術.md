# 《暗線 / DARKLINE》首部曲 — 年代考據：服裝・用品・武器・美術風格（1953 台北）

> 產出日期：2026-06-17 ｜ 性質：**美術／資產考據參考**（用戶要求：服裝、用品、武器、美術風格全部對齊年代；**2.5D 非全擬真，但要有年代沉浸感、讓各年齡玩家驚豔**）。
> 配套：[1953 台北街景考據](DARKLINE-1953台北街景考據.md)（場景）、[市場研究](DARKLINE-市場研究報告.md) §6/§7（既有服裝/武器底）。

## 0. 沉浸感的兩個讀者：為什麼要做對

- **年長玩家＝辨識的鄉愁**：搪瓷臉盆、三輪車、真空管收音機、旗袍、暖水瓶——他們**認得**，一眼回到童年＝驚豔來自「對」。
- **年輕玩家＝陌生的酷**：一個沒見過的世界，被風格化得很帥（boomer-shooter 的可讀性＋琥珀 noir 調）＝驚豔來自「新又美」。
- **橋樑＝風格化讓年代變酷、不是變博物館**：細節要真，呈現要 stylized。下面每節都先給「真實」再給「2.5D 怎麼做」。

## 1. 服裝（依陣營／階層／性別；剪影可讀 ＝ sprite 第一要務）

> 1953 台北的服裝＝**旗袍×西式×軍裝×布衫**並存的過渡風貌；社會氛圍是政府宣導**黜奢崇儉**（樸素布衣布鞋），但上層與影視圈仍有旗袍華風。

| 類型 | 真實質地 | 2.5D sprite 做法 |
|---|---|---|
| **本省平民・女** | 旗袍／碎花洋裝、布衫、木屐或皮鞋、燙髮波浪或盤髮 | 旗袍剪影（立領、開衩）＝最強年代符號；碎花用 2~3 色塊近似 |
| **本省平民・男** | 對襟衫／唐裝、工人布衫、草帽或呢帽（fedora）、布鞋 | 寬鬆布衫＋呢帽剪影；勞動者捲袖 |
| **外省人／上層** | 旗袍（蔣宋美齡帶動的雙圓襟一度流行）、西裝、呢帽 | 較挺的西裝剪影、女性較華的旗袍配色 |
| **公務／官署（內勤科）** | 中山裝、西裝、戴帽；冷硬、整齊 | **深色西裝／中山裝＋呢帽＝「內勤科」便衣的可怕在於不顯眼**（見 §4 武器） |
| **國軍** | 卡其／草綠軍裝、大盤帽或船形帽、綁腿 | 軍裝剪影＋帽型分敵我 |
| **駐台美軍** | 卡其制服、軍便服、士兵與軍官 | 美式剪影＋膚色/制服色區分 |
| **學生** | 制服（卡其／白衫黑裙）、書包 | 自由段背景人群用 |
| **將軍的美械新軍（友方）** | 美式鋼盔、美械裝具 | 與國軍舊裝、北方軍裝三方剪影對比 |

> ⚠️ **中國/台灣混淆紅線（搜尋常混入）**：「**布拉吉**」（俄式連衣裙）、「中山裝＝資產階級情調而消失」是**中國大陸 1950s（蘇聯影響/革命語境）**，**台灣沒有這套**——台灣是旗袍華風＋西化，**勿套用**。

**伏筆視覺鉤**：投誠者／義士的**軍便服＋可見刺青**（反共標語）＝首部曲規劃既定的視覺伏筆，保留。

## 2. 日常用品／道具（鋪滿街景與室內＝沉浸感的底）

> boomer-shooter 的「豐富」來自**小道具密度**，不是高模。以下每樣都是低面數 primitive＋貼圖即可，越多越有年代味。

- **居家**：搪瓷臉盆／杯（常印雙喜或標語）、**暖水瓶**（玫瑰花紋鐵殼）、真空管收音機（木殼、布網喇叭）、縫紉機（鑄鐵腳踏式）、藤椅、蚊帳、煤油燈、灶。
- **街頭**：三輪車（帆布頂）、腳踏車、麵攤／切仔麵擔、冰果室、香菸攤、報攤、布條標語、曬衣繩、電線桿。
- **辦公／官署（內勤科質地）**：打字機、算盤、鋼筆、印章、**公文夾與卷宗**、檔案櫃、手搖／轉盤電話、橡皮圖章與紅印泥——**「查封公文」「逮捕令」這些紙本道具＝本作官僚之惡的視覺語言**（接劇情）。
- **商業**：茶箱、麻布袋、藥櫃（大稻埕）、月份牌（月曆美女海報）、手繪招牌、霓虹燈管。

> ⚠️ **品牌別亂掛**：「紅燈牌收音機」「飛鴿牌腳踏車」是**中國大陸品牌**；台灣 1950s 腳踏車有幸福牌／伍順等。**不確定就做無牌**（招牌/字樣留白或虛構），免時代+地域錯置。

## 3. 諜報道具（接你已做的電報／解碼／死信箱美術）

> 本作是諜報片，tradecraft 道具＝克蘭西式「真實感」的關鍵，且**接得上現有的解碼面板（琥珀電報調）與死信箱紙片梗**：

- **電報／電碼**：摩斯鍵、電報紙條、密碼本／一次性密碼表——**直接呼應已做的 #overlay 琥珀電傳打字機美術**，當全片母題延伸。
- **相機**：雙眼相機（TLR，如 Rolleiflex 形）、微縮膠卷＝跟監／蒐證。
- **藏匿**：香菸盒／火柴盒夾紙條、報紙夾傳遞、死信箱（牆縫、廟埕磚孔）＝**死信箱紙片＝鑰匙**的實體依據。
- **文書**：鋼筆、打字機打的密件、騎縫章——與 §2 官署道具同一套，呼應「名單被做成證物」。

## 4. 武器（陣營對比＝劇情的視覺論證）

> 軌道射擊不需武器庫龐大；重點是**用武器剪影講陣營與主題**。本作的關鍵主題＝「**你怕的敵人拿衝鋒槍，真正做掉你的人拿一把藏在西裝裡的左輪**」。

| 陣營 | 武器（史實對應） | 主題作用 |
|---|---|---|
| **林建國（玩家）** | **M1911A1** .45 手槍（沿用現有 view model） | 美式裝備＝西緣/美方體系 |
| **將軍美械新軍（友方）** | M1 Garand、M1/M2 卡賓槍、Thompson／M3「Grease Gun」、BAR | 最現代的美械＝將軍「太美國」被猜忌的視覺由來 |
| **「內勤科」秘密警察（真‧反派）** | **美製 .38 左輪／藏於西裝的 M1911**；低階用遺留**南部十四年式**日本手槍 | **不顯眼的便衣＋一把安靜的手槍＝最致命**——比北方衝鋒槍更可怕，扣本作「自己人」之惡 |
| **北方滲透網（明面敵人／Boss）** | **PPSh-41／Type 50「burp gun」**（彈鼓）、TT-33 托卡列夫、Mosin-Nagant | 強烈的蘇式剪影＝玩家「以為的敵人」（其實是煙霧的一半） |
| **背景／雜兵・遺留日械** | **三八式步槍**（1950s 台灣多為教練／刺槍術用）、南部手槍 | 戰後過渡質地；當背景道具與低階武裝 |

> ⚠️ **時代註記**：三八式 1950s 在台已退出國軍實戰一線（多教練用，後由 57 式取代）→ 當**背景／雜牌**質地，別當主戰武器。美製 .38 左輪是 ROC 軍警常見配槍質地（型號可虛化）。武器外觀**參考但原創化處理**（型號是公共領域史實，避免照搬特定品牌商標細節，沿用規劃 §7 守則）。

## 5. 2.5D 美術風格：年代沉浸感怎麼做（非全擬真）

> 本作＝**風格化 2.5D**（unlit billboard sprite＋簡單 3D 環境＋程式化調色盤），不追 PS3 擬真。沉浸感靠「**對的細節＋對的色調＋對的顆粒**」，不靠面數/解析度。

1. **調色＝琥珀暖陽 × 青冷陰影**：沿用既有 `DARKLINE_PALETTE`（24 色降級收斂風格）＋夕陽暖調（用戶已認可的 Duke3D 街景調性，keeper）。整體往「**上色老照片／月份牌**」的暖褐＋局部青綠走＝一眼就是那個年代。**琥珀電報調**（解碼/overlay 已起頭）當**全片視覺母題**延伸到 HUD/字卡。
2. **顆粒＝老膠片**：bloom／grain／vignette／輕色調分離（M3 後處理，spec §8）＝最便宜、效果最大的「年代沉浸」開關。
3. **密度＝boomer-shooter**：用 §2 的小道具／招牌／海報**鋪滿**（Duke3D/Ion Fury 路線），密度＝豐富感；幾何維持輕量、細節交給貼圖與貼花（decal）。
4. **剪影可讀＝sprite 鐵則**：陣營/階層一眼分（旗袍平民／西裝左輪內勤科／彈鼓北方／鋼盔新軍），即 §1/§4 的剪影對比。
5. **招牌與海報**：直式中文手繪招牌、月份牌美女、西門町手繪電影海報、中山北路英文招牌＝年代質地的免費大補帖（生圖貼上幾何）。
6. **驚豔點＝風格化的真實**：細節要讓年長玩家「認得」（搪瓷、三輪車、旗袍），呈現要讓年輕玩家覺得「酷」（noir 琥珀光、爆裂槍感、乾淨剪影）。

**Gemini 生圖守則**（沿用）：避「VC 風格」字樣與 SEGA 角色；套調色盤；只貼程序幾何不追解析度；無牌或虛構招牌避免時代/地域錯置。
