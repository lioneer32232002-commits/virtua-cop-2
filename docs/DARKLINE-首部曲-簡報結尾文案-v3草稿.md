## 1. 開場簡報（西緣貿易公司閣樓・城內官署區・入夜）

> v3 §6 拍1：老聶帶林建國受命——名單外勤接連失聯，查獵殺、堵洩密（玩家此刻以為是北方）。tradecraft 開場。**埋種子①**：老聶輕點名字真意，不解釋。
> 形制：**2 頁**（Page 1 建立世界與 tradecraft；Page 2 下任務＋埋種子）。

### Page 1／2 — 這行怎麼運作（先讓你懂規矩）

```
一九五三年　台北　入夜

西緣貿易公司　三樓閣樓
表面上，我們替美國人算船期、報關、押貨。

實際上，這間閣樓是一張網的結點——
死信箱、暗碼本、聯絡的規矩，
把城裡看不見的人一條線串起來。

名冊上每一個代號，都是一條命。
規矩只有一條：名字不落紙，人不見面。
```

### Page 2／2 — 受命（埋種子①）

```
師父老聶把一疊回條推過來。

「外勤的燈，一盞一盞滅了。」
「三個星期，五個代號，沒回音。」

他沒抬頭：「北方的人也許進了城。
查清楚誰在獵我們的人，把洩的口堵上。」

臨出門，他忽然叫住我，
看著我識別證上的名字，很輕地說：

「建國……你這名字，取得真好。」
「愛國。」他頓了一下，「大家都這麼聽。」

我沒接話。
有些字，只能在心裡讀成另一個樣子。
```

> 【種子①】老聶「只有他懂那個國」的伏筆＝此處**唯一**明點、且**不解釋**（§2 鐵律）。「大家都這麼聽」＝雙關的門，開一條縫即關。
> 【接續】末頁 → Arena 1（榮町騎樓街）。

## 2. 中段字卡（可選・自由段私密鏡頭＝種子②）

> v3 §2 種子②：中段一個私密鏡頭（望河/島一眼、信裡半句話），點到不解釋。做成**自由段可選觸發**（走近河堤/窗邊觸發一張字卡），不強塞。

```
（大稻埕　河堤邊　可選）

淡水河往北流。
對岸的燈，這幾年我學會了不去數。

老聶說，人要認命。
可我認的那個，說出來就是死。

河水不問你效忠誰，只管往海裡去。
```

> 【種子②】只此三行、無旁白解釋。玩家路過即得，錯過不影響主線＝「私密」質地。

## 3. 結尾（將軍線收束＋搶救真相＋1996 鉤子）

> v3 §6 拍6：將軍升官架空→軟禁（不露面、將軍玫瑰遠景）；克蘭西式勝利（搶救真名單＋刑求鐵證送進未來死信箱）；種子③收束（把真相與夢一起無聲交給後人）；1996 鉤子。
> 形制：**2 頁**（Page 1 悲劇落幕＋將軍玫瑰；Page 2 搶救真相＋種子③＋1996 鉤子）。結尾台詞採 v3 §8 #2 題眼、#4 接 1996。

### Page 1／2 — 救不回的（將軍玫瑰・不露面）

```
名單守住了。人，沒有。

老聶簽下的那張「口供」，
成了整套案子的地基——
他們用一個好人的字，埋掉了一串好人。

城郊有座宅子，將軍被榮譽的虛職架空，
如今由衛兵看著。
聽說夫人在後院種玫瑰，拿到市場去賣。

一個打過三國勳章的人，
最後輸給的不是敵人，
是自己人怕他不必靠他們、也照樣能贏。
```

> 【演出】將軍全程不露面：以旁白/遠景/轉述帶出；「將軍玫瑰」遠景一瞥＝最強一擊。

### Page 2／2 — 送進未來的（種子③＋1996 鉤子）

```
我把真名單，連同「口供是刑求逼出來的」鐵證，
封進一個只有將來打得開的死信箱。

將軍留給我一句話，我譯成暗碼收進去：
「名字留住。人，他們帶得走；名字，要靠你帶走。」

在被劃掉的代號旁，我又添了一行，
一半給後來的人，一半——給那個我沒說出口的國：

　　　　　待第一島鏈再次收緊之日

從今夜起，名冊上不再有我的名字。
他們會叫我，暗線。
```

```
　　　　　　　　　—— 一九九六　台海　續 ——
```

> 【題眼】將軍台詞 #2 咬合林建國名字（「名字要靠你帶走」）＝雙關收束。
> 【種子③】「一半給那個我沒說出口的國」＝隱性靈魂最後一次浮現，仍不解釋、無口號。
> 【1996 鉤子】沿用既有伏筆字串「待第一島鏈再次收緊之日」→ 銜接第二部曲。

### Opening — Page 1／2

```
TAIPEI — 1953. AFTER DARK

Third floor, West-Marches Trading Co.
On paper, we clear cargo for the Americans —
manifests, customs, freight.

In truth this loft is the knot in a net:
dead drops, cipher pads, the rules of contact,
tying together the men this city never sees.

Every codename in the book is a life.
One rule only: no name on paper, no face in the open.
```

### Opening — Page 2／2（種子①）

```
Old Nieh pushes a stack of return-slips across the table.

"The outer lamps are going dark. One by one."
"Three weeks. Five codenames. Silence."

He doesn't look up. "The North may be inside the city now.
Find who's hunting our people. Close the leak."

At the door he stops me,
his eyes on the name on my papers, and says, softly:

"Chien-kuo. 'Found a nation.'" A pause.
"A fine name. Everyone hears it the same way — a loyal son of the Republic."

I said nothing.
Some words you can only read, in private, as another country.
```

The list was saved. The men were not.
