## 0. 一句話故事（v2，克蘭西調）

> 1953 台北。林沂——情報網裡一個不起眼的**密件聯絡員／譯電手**——發現名單上的外勤一個個「消失」。他以為是北方下手，沿著電文與死信箱往上追；真相是**自己人**：有人為了扳倒他們效忠卻從未謀面的**戚定遠將軍**（美式軍校英雄、因美方賞識而被體制猜忌的「外人」），正把名單上的人逐一**構陷成「匪諜」**，連他敬重的師父老聶都被抓去逼供、簽下自誣的口供。林沂救不了師父、也擋不住將軍被架空軟禁——**但他能趕在機器把一切抹平之前，把構陷的鐵證和真名單送出去。** 人他救不回，真相他守得住。

**這是一個「能人小人物揭穿體制內構陷」的悲劇＋克蘭西式硬底子諜報**：動作骨架（軌道→自由→Boss）不動，玩的還是同一款射擊，但隨進度，玩家會發現槍口外的真相一直在移位——敵人不在街對面，在自己後方。

## 1. 邏輯檢查與我做的修正（用戶要求）

> 你要我檢查邏輯、不好玩就改。以下是把你四個選擇疊在一起後的風險，與我的處置。

**風險：** 1B（基層主角）＋師父救不回＋將軍不露面＋全面背叛，若配**純 le Carré 悲劇**，會變成「**一個都救不了、一個都贏不了**」＝無力到不好玩，也**不是克蘭西**。

**修正（核心）：把主角的勝利條件從「救人」改成「保住真相」。**
- 林沂救不了師父、阻不了將軍被軟禁（悲劇保留）——**但他做得到「搶救鐵證＋真名單」**。這是可達成、有爽感的玩家目標，且**呼應史實**：正是被偷偷保住的真相，幾十年後促成遲來的平反。
- **1B 因此變優點**：越基層、越被陰謀者忽視，越能在暗處把線索連起來＝**Jack Ryan 式「被低估的分析腦」**。林沂的武器不是官階，是**眼力與韌性**。
- **Boss 升級成真‧勁敵**：北方的行動是真實、危險、必須阻止的；但內勤科**利用**這場仗，把林沂的網「順勢」做成通敵鐵證。→ 林沂**打真敵人、卻發現自己人在拿這仗害自己人**＝克蘭西地緣實感 ＋ le Carré 內部背叛。

**克蘭西調性的落地手段（融進設計）：**
- **專業實感（tradecraft）＝賣點**：死信箱、譯電、跟監偵防、查封公文的紙本軌跡——讓「情報工作怎麼真的運作」變成玩法質感（且接得上**已做好的解碼／死信箱紙片系統**）。
- **能人平凡主角**：林沂不是神槍特工，是會判讀、肯查、敢追的普通人。玩家的成就感來自「**我把別人沒看出來的連起來了**」。
- **地緣框架**：駐台美軍／CIA 前線／北方滲透是真實舞台，賭注是國家級的，不是私人恩怨。
- **比 le Carré 多一分熱血**：結局慘重但**真相出鞘**、種子留下——不是虛無，是「代價高昂的正確」。

## 2. 戚將軍 ＝ 孫立人原型的全面虛構化（將軍「只聞其人」）

> 左欄＝真實原型質地（僅靈感）；右欄＝遊戲虛構化。**將軍全程不露面**，只透過電文、他人轉述、與留下的暗記存在——更諜報、更留白、也更突顯「林沂效忠一個自己幾乎不認識的人」的克蘭西式體制感。

| 真實原型（僅靈感） | 遊戲虛構化（戚定遠將軍） |
|---|---|
| 留美工科→棄筆從戎入美國軍校；非黃埔嫡系 | **美式腦袋、不結派的孤臣**；靠真本事不靠關係 |
| 緬甸救七千盟軍＋三國授勳 | **異域戰功、多國授勳**＝國際公認英雄、美方公開賞識 |
| 鳳山練美械新軍 | 在台練一支**最現代、最忠於他個人**的美械新軍 |
| 升官架空→撤職→軟禁 33 年 | **升官架空成榮譽虛職 → 構陷案後軟禁**，特務監視 |
| 軟禁時夫人種玫瑰拿到市場賣（「將軍玫瑰」） | **將軍玫瑰**母題：英雄晚景宅院賣花（結尾最強一擊，以旁白/遠景帶出，人不露面） |

**為何是金礦**：①「太美國而被自己人猜忌」天生扣本作副標 First Island Chain；②害他的人未必是壞蛋，是真心信「不聽話又被美國人捧的將軍遲早是禍」＝**平庸之惡**；③§13 最安全——批判**主題**（體制容不下不結派的英雄），不指名任何人/黨。

## 3. 角色表（v2：林沂＝基層、老聶＝師父，名可換）

- **林沂／代號「暗線」**（主角）：情報網最末梢的**密件聯絡員兼譯電手**——跑死信箱、解暗碼、登記名單往來。官階低到陰謀者懶得防他，**卻是唯一天天碰「名單流量」、最先察覺名字一個個沒了的人**。沒有靠山、沒有官威，只有眼力和一股不肯放手的勁。**「暗線」這個傳奇代號，是他在這場劫難裡掙出來的**——本作＝他的起源故事（為 1996／當代部曲傳承鋪路）。
- **老聶**（**悲劇支點＝師父**，郭廷亮原型，名可換）：林沂的引路人、將軍的忠誠老士官，教會林沂這行的規矩與底線。名單上**第一個被抓**的人。在 Arena 2 **救不回**（見 §5）——被帶走、刑求、簽下自誣為匪諜的口供。那份假口供成了整套構陷的**基石**：有了它，名單上每個名字都能被「順藤摸瓜」做成將軍的通敵網。**他是玩家的情感錨**（將軍不露面，痛由師父扛）。
- **站長蕭敬之**（**proximate 背叛者**）：林沂在西緣的上線、老聶被抓後林沂唯一還剩的「長官」——而他正是把名單暗中交給內勤科的人。早看出將軍要倒、押注得勢一方，順手在查封的財產裡分贓（掺候選 A「分紅」的私心，讓背叛更髒更真）。**「老聶被抓後，林沂能依靠的最後一個人，就是出賣大家的人」＝克蘭西／le Carré 最愛的孤立感（誰都不能信）。**
- **戚定遠將軍**（精神核心，**只聞其人不露面**，見 §2）。
- **「內勤科」**（虛構內安機構，**用戶確認沿用**）：扳倒將軍的派系機器；1953 國安局未成立故用虛構名、避時代錯置。蕭敬之只是它伸進西緣的一隻手。
- **北方行動指揮**（Boss，**真‧勁敵**）：一個真實、危險、值得敬畏的對手——林沂必須阻止他的行動；但內勤科**利用**這場交火，把它做成「林沂的網通敵」的證據。打贏他，只解決了一半，另一半在你背後。

## 4. 麥高芬：名單從「被搶」變「被當證物／被搶救的真相」

- 舊：名單＝北方要搶。**新：名單＝自己人要拿來「做案子」的證物——也是林沂要搶救的真相。**
- 將軍的新軍太現代、太得兵心、又被美方公開賞識→在不結派者眼中＝最大政治威脅。動不了將軍本人→**先拆他的網**：把名單外勤逐一構陷成「匪諜」，沒收、收押、逼供（用上**檢舉分紅＋連坐聯保**的真實制度土壤，見候選文件原型②），層層做成「戚將軍部下集體通敵」大案→好把將軍架空軟禁。
- 被「劃掉」的人**不是北方殺的，是自己人「結算」掉的**。林沂的任務從「找出殺手」翻轉成「**證明這些人是被製造成叛徒的，並把真名單與鐵證送出去**」。
- → 麥高芬從「諜報動作」（搶情報）升級成「諜報悲劇」（情報如何被自己人拿來殺自己人），**而玩家的手，握著把真相帶出去的最後機會**。

## 5. 六拍串接（動作骨架不動、重寫意義；套用你的選擇）

1. **開場簡報（西緣閣樓）**：老聶帶著林沂受命——名單上的外勤接連失聯，查是誰在獵殺、堵洩密。（玩家此刻以為是北方。）**tradecraft 開場**：交代死信箱、譯電、聯絡規矩＝克蘭西式「先讓你懂這行怎麼運作」。
2. **Arena 1（騎樓街）**：便衣伏擊、軌道清場。林沂追第一條死亡線索。**第一個疑點**：死者不是被搶情報，是**家產被「依法」查封**——不像北方手法。
3. **Arena 2（市場／巷口・人質拍）＝師父救不回**：要保護的人是**老聶**。交火中——**真正帶走老聶的不是北方，是趁亂行動的內勤科**（玩家忙著打北方槍手，等清完場才發現師父已被「依法逮捕」帶走）。**救不回**：不是被一槍打死，是被體制「合法」吞掉——更冷、更克蘭西。這一拍埋下「假口供成鐵證」的種子，也是林沂第一次嚐到「敵人在背後」。
4. **自由段（調查／解碼，接已做系統）＝Jack Ryan 把點連起來**：
   - 比對「**死亡/失蹤名單** vs **沒收·收押公文**」→ 死亡順序＝結算順序，手法全是**自己人的官僚動作**（查封、檢舉、逮捕）。
   - 追洩密源 → 線頭**指回西緣內部、指回蕭敬之**。
   - **死信箱紙片＝鑰匙**（沿用現有梗）：改寫成**老聶被押走前偷塞出的口供碎片／暗記**；解碼解出的不是交接地點，而是「**口供是逼出來的、名單還在我們自己手上**」這句真相。＝玩家「親手解出陰謀」的爽點。
5. **Arena 3 ＋ Boss（碼頭／美軍區外圍）＝真敵人，假勝利的一半**：林沂阻止北方行動指揮的真實行動（**正當的、緊張的、值得打的**一仗）。**轉折**：打贏才發現，這場交火被內勤科做成了「林沂的網與北方接觸＝通敵」的鐵證——**他親手替構陷補上了最後一塊拼圖**。勝利是空的一半，另一半在身後。
6. **結尾（將軍線收束＋搶救真相＋1996 鉤子）**：
   - 將軍被**升官架空→軟禁**，以**旁白／遠景／他人轉述**帶出（不露面）；**將軍玫瑰**母題（宅院賣花的英雄，遠景一瞥）。
   - **林沂的克蘭西式勝利**：他搶在卷宗封存前，把**真名單＋「口供是刑求逼供」的鐵證**送進一個只有未來才打得開的死信箱——人救不回，真相留住了。
   - 名單上被劃掉的代號旁，他留下那行字「**待第一島鏈再次收緊之日**」＝把真相託付給未來：島嶼總有再被收緊的一天，到那天，先回頭看自己人。
   - 將軍的平反**遲了一輩子**（接 1996／當代部曲）；林沂自此遁入暗處，成為「暗線」。

## 7. 結尾將軍台詞（item ⑤，由我擬；將軍不露面→以轉述/暗記/留書送達）

   —（老聶在被帶走前，替將軍轉述給林沂；點明「忠誠被當成罪」的核心。）
2.「**名字留住。人，他們帶得走；名字，要靠你帶走。**」
   —（將軍透過死信箱留給林沂的一句託付；直接對上「搶救名單」的玩法與麥高芬。）★最推薦當全片題眼
3.「他們不是怕我造反——他們是怕一個不必靠他們、也照樣能贏的人。」
   —（自由段查到的一份將軍舊信殘頁；點破「外人英雄」被猜忌的真因。）
4.「告訴後來的人：島嶼會再被收緊。到那天，先回頭看看自己人。」
   —（林沂把它譯成暗碼、寫進留給未來的名單旁；無縫接 1996／當代部曲鉤子。）

## 8. §13 合規自查（v2）

- ✅ **不指名**：戚將軍＝孫立人原型而非孫立人；無國民黨/黃埔/真實機關（用虛構內勤科）；不指名任何真實在世/已故政治人物。
