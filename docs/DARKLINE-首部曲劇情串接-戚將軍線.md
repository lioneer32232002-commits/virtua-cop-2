## 0. 一句話故事

> 1953 台北。情報員林沂替他敬重的**戚定遠將軍**（美式軍校出身、緬甸戰功彪炳、卻因美方賞識而被體制猜忌的「外人」名將）守著一張本土情報網的「暗線名單」。名單上的人一個個被獵殺——林沂沿線追兇，以為是北方下的手；真相是**自己人**：有人為了扳倒這位不結派、功高震主的將軍，正把名單上的外勤一個個**構陷成「匪諜」**，層層做成「將軍的部下通敵」的案子。林沂能贏每一場槍戰，卻救不了將軍、洗不清被製造出來的「叛徒」——他守住了名單，守不住「我們是好人」這件事。

**這條線把候選 B 的抽象機制（自己人構陷自己人）落到一個具體的、玩家會心疼的人身上**——將軍。背叛因此不是新聞案件，是主角親歷的傷。精神標竿＝le Carré（體制內背叛比敵人致命）＋《Papers, Please》（小人物被體制碾過）。

## 1. 戚將軍 = 孫立人原型的全面虛構化

> 規則：下表左欄是**真實原型質地**（靈感），右欄是**遊戲裡的虛構改寫**。遊戲只出現右欄，不指名孫立人。

| 真實原型（僅靈感） | 遊戲虛構化（戚定遠將軍，placeholder 名可換） |
|---|---|
| 清華庚款留美→普渡土木工程→棄工從軍入維吉尼亞軍校 | 留美工科生棄筆從戎、進**美國軍校**——徹底的「美式腦袋」，不是本土軍系出身 |
| 非黃埔嫡系；早年帶稅警總團等「雜牌」 | **不屬任何派系**的孤臣；靠真本事不靠關係上位 |
| 緬甸仁安羌救七千英軍＋英軍司令；蔣/羅斯福/英王三國授勳 | 二戰**異域戰功**救過盟軍、受**多國授勳**＝國際公認的英雄；美方公開賞識 |
| 印度蘭姆珈美械訓練→新一軍；1947 鳳山練美械新軍 | 在台**鳳山式基地練一支美械新軍**＝最現代、最忠於他個人的部隊 |
| 陸軍總司令→總統府參軍長（升官架空）→1955 撤職軟禁台中 33 年 | 被**「升官架空」成榮譽虛職**→構陷案後**軟禁**，特務監視、不得與人往來 |
| 軟禁時夫人種玫瑰拿到市場賣（「將軍玫瑰」） | **將軍玫瑰**母題：戰功彪炳的英雄，晚景是宅院裡賣花（結尾最強的視覺一擊） |
| 1988 解禁、遲來平反；2001 監院認部屬非匪諜 | 平反**遲了一輩子**＝1996/當代部曲的伏筆 |

**為什麼這個原型對 DARKLINE 是金礦：**
1. **「美式教育卻被打入冷宮」這條張力天生扣題**——本作副標就是 First Island Chain、世界觀就是駐台美軍與 CIA 前線。一個「太美國」而被自己人猜忌的將軍，把「美台關係的甜與毒」濃縮成一個人。
2. **道德灰度純度最高**：害他的人未必是壞蛋——他們真心相信「一個不聽話、又被美國人捧著的將軍，遲早是禍」。**平庸之惡**，不是臉譜反派。
3. **§13 最安全**：批判的是「**體制容不下不結派的英雄、用構陷整肅異己**」這個**主題**，不指名任何政黨/人物（孫只是原型）。

## 2. 角色表（虛構，placeholder 名可換）

- **林沂／代號「暗線」**（主角，沿用）：戚將軍**緬甸時帶出來的舊部**，如今掛「西緣貿易公司物流顧問」的皮（CIA 前線掩護），實為**替將軍與美方共管那張本土情報網的人**。忠於將軍、信「我們是好人」——這份信念是這條線要一刀一刀割掉的東西。
- **戚定遠將軍**（精神核心，見 §1）：林沂效忠的對象。**全程幾乎不開槍**——他的戰場是被構陷的審判桌，不是街頭。玩家為他而戰，卻救不了他。
- **站長蕭敬之**（**proximate 背叛者**，沿用候選文件）：林沂在西緣的**上線**。早就看出將軍要倒、押注在得勢的一方，**把暗線名單暗中交給「內勤科」**，順手在沒收的財產裡分一杯羹（＝掺入候選 A「分紅」的私心，讓背叛更髒更真）。最不該背叛的人，背叛了。
- **老聶**（**悲劇支點**＝郭廷亮原型，名可換）：林沂緬甸時的**生死弟兄**、將軍的忠誠老士官。名單上第一個被抓的人——被刑求逼供、簽下**自誣為匪諜的口供**。那份假口供成了整套構陷的**基石**：有了它，名單上每個名字都能被「順藤摸瓜」做成將軍的通敵網。
- **「內勤科」**（虛構內安機構，沿用）：1953 時點負責「檢肅」的安全單位虛構代稱（國安局 1955 才成立，故不用真名、避時代錯置）。扳倒將軍的**派系機器**，蕭敬之只是它伸進西緣的一隻手。
- **北方行動指揮**（原 Boss，意義被重寫）：玩家一路以為的「敵人」——但他要麼是**煙霧**，要麼根本是被內勤科**反過來栽贓**、好坐實「將軍通敵」的道具。打贏他，毫無快感。

## 3. 麥高芬重定義：名單從「被搶」變「被當證物」

舊設定：暗線名單＝北方要來搶的東西。
**新設定：暗線名單＝自己人要拿來「做案子」的證物。**

- 將軍的鳳山新軍太現代、太得兵心、又被美方公開賞識→在不結派的人眼裡＝**最大的政治威脅**。
- 動不了戰功彪炳的將軍本人→**先拆他的網**：把暗線名單上的外勤逐一構陷成「匪諜」，沒收、收押、逼供（用上**檢舉分紅＋連坐聯保**的真實制度土壤，見候選文件原型②），再層層往上做成「**戚將軍的部下集體通敵**」的大案——最後好名正言順地把將軍**架空、軟禁**。
- 名單上被「劃掉」的人，**不是被北方殺的，是被自己人「結算」掉的**：死的死、關的關、被逼成假匪諜的成了「鐵證」。

**麥高芬於是從「諜報動作」（搶一份情報）升級成「諜報悲劇」（一份情報如何被自己人拿來殺自己人）。**

## 4. 對既有六拍結構的串接（對映 [首部曲規劃](DARKLINE-首部曲規劃.md) §2.51）

> 動作骨架（軌道→自由→Boss）不動，**只重寫每一拍的「意義」**——玩家玩的還是同一款射擊，但隨著進度，槍口外的真相一點點移位。

1. **開場簡報（西緣閣樓）**：林沂受命於將軍/西緣，守住一張**正在被「逐一抹除」的名單**。表面任務：找出是誰在獵殺外勤、堵住洩密。（玩家此刻以為是北方。）
2. **Arena 1（騎樓街）**：便衣伏擊、軌道清場。林沂追第一條死亡線索。**第一個疑點**：被殺的外勤死前不是被搶情報，是**家產被「依法」查封了**——不像北方手法。
3. **Arena 2（市場／巷口・人質拍）**：要保護的「平民」＝**老聶**（或唯一還能指證構陷的證人）。交火中保住他＝保住真相的最後機會。**若沒保住**（或劇情上他已被帶走）→ 他的假口供成案，全盤皆輸的種子埋下。
4. **自由段（調查／解碼，接已做系統）**：
   - 比對「**死亡/失蹤名單** vs **沒收·收押清冊**」→ 死亡順序＝結算順序，手法是「檢舉、查封、逼供」，**全是自己人的官僚動作**。
   - 追洩密源 → 線頭**指回西緣內部**、指回蕭敬之。
   - **死信箱紙片＝鑰匙**（沿用現有梗）：改寫成**老聶在被押走前偷遞出的口供碎片／暗記**——解碼解出的不是交接地點，而是「**口供是逼出來的、名單在我們自己手上**」這句真相。
5. **Arena 3 + Boss（碼頭／美軍區外圍）**：追上「交接點」、打北方行動指揮。**轉折**：這場仗是煙霧——林沂打贏了，才發現名單早就在自己人手裡轉了一圈，將軍的案子已經做成。**真正的反派從不開槍。**
6. **結尾（將軍線收束＋1996 鉤子）**：
   - 演出將軍被**升官架空→軟禁**；**將軍玫瑰**母題（宅院賣花的英雄）。
   - 林沂最後一次見將軍，將軍一句**改寫自史實精神、非照搬原句**的話（例：「我這一生沒有反過誰，他們卻說要『平』我的反」——**原創句、可再潤**）。
   - 名單上被劃掉的代號旁那行字「**待第一島鏈再次收緊之日**」＝林沂留的暗記：島嶼總有再被收緊的一天，而那時真正的威脅一樣可能來自**內部**。
   - 將軍的平反**遲了一輩子**（接 1996／當代部曲）。下一代繼承了暗線的技藝，也繼承了這道傷：**敵人，可能在我們之中。**

| 拍點 | 玩家以為 | 真相 | 情緒 |

## 6. §13 合規自查

- ✅ **不指名**：戚將軍是孫立人**原型**、不是孫立人；不出現國民黨/黃埔/任何真實機關（用虛構「內勤科」）；不指名任何真實在世/已故政治人物。
- ✅ **主題式影射，非對象式**：批判「體制猜忌容不下不結派的英雄、用構陷整肅異己」這個**主題**；不是對某政黨的控訴。
- ✅ **道德灰度、非政治表態**：害將軍的人有其「為黨國除患」的扭曲邏輯＝平庸之惡，不是反共/反某黨宣傳。
- ✅ **片頭免責**「純屬虛構」沿用。
- ⚠️ **史實精神句要原創改寫**，不照搬孫立人原話（如「從未反過，何平之有」這類），避免可被指認的直引。

## 7. 待你判對味的幾個鈕

1. **主角與將軍的距離**：
   - (a)【建議】林沂＝將軍**緬甸舊部、暗線網的主事者**＝「旁邊的人」裡離將軍最近、權重最高的一種（背叛最痛、也最能扛主角戲份）。
   - (b) 林沂＝更**基層的副官/聯絡官**，悲劇支點老聶才是他的師父＝更純的「小人物被巨輪輾過」（Papers Please 味更重，但主角推動力較弱）。
2. **老聶（郭廷亮原型）的下場演出**：Arena 2 救得回/救不回？「救不回」最悲、最扣「假口供成鐵證」；「救回但已簽供」更殘忍（救了人卻救不了已成的案）。
3. **將軍是否在遊戲中現身**（過場/簡報立繪）還是**只聞其人不見其面**（更像諜報、留白更有力）？
4. **名字**：戚定遠／老聶／站長蕭敬之／內勤科——可用或想換？
5. **結尾將軍那句話**：要我多擬幾個原創候選句給你挑嗎？
